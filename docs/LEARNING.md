# Aprendizaje transparente

Fuentes: tareas completadas o pospuestas, duración real, sesiones de enfoque y revisión diaria.

- Duración: 70% estimación previa + 30% duración observada.
- Carga: 85% cuando el día fue irreal o el cumplimiento quedó bajo 60%.
- Bloques: 35 minutos con energía baja, 50 media y 75 alta.
- Descansos: 15 minutos con valoración ≤2 o plan irreal; 10 en otro caso.
- Fatiga: proporción de revisiones con energía baja o plan irreal en las últimas diez muestras.

No se usa machine learning opaco.

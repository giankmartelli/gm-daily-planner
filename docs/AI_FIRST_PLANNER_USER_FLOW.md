# Flujo de usuario

1. **Analizar:** recopilar tareas, agenda, objetivos e historial disponibles. Fuentes ausentes se muestran como tales.
2. **Proponer:** crear cambios estructurados sin modificar la agenda.
3. **Revisar:** mostrar horarios, motivo, beneficio, riesgo, confianza y procedencia.
4. **Ajustar:** aceptar, modificar o rechazar cambios individuales.
5. **Confirmar:** resumir el resultado y solicitar “Aplicar plan”.
6. **Ejecutar:** aplicar una sola vez, persistir y sincronizar.
7. **Deshacer:** restaurar el snapshot anterior.
8. **Aprender:** registrar una revisión transparente al terminar el día.

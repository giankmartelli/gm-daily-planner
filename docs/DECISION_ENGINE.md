# Decision Engine

```text
score = prioridad × .24 + urgencia × .25 + impacto × .22
      + energía × .10 + duración × .11 + dependencias × .08
```

El resultado se limita a 0–100. Las vencidas reciben urgencia 100; la ausencia de fecha no se interpreta como vencimiento. Cada resultado incluye el desglose completo.
